## How to install ?

~~ cd darkweb

~~ pip3 install -r requirements.txt

~~ chmod +x build.sh

~~ ./build.sh



## Usage

~~ python3 darkweb.py

DarkWeb v3.0 usage:

`  -s      --start        Start DarkWeb`

`  -r      --switch       Request new tor exit node`

`  -x      --stop         Stop DarkWeb`

`  -h      --help         Print this help and exit`

`  -u      --update       Checks for updates`
